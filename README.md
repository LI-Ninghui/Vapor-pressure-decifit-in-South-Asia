# Vapor-pressure-decifit-in-South-Asia
VPD trend, 2-D pattern in CMIP6/LESFMIP/ScenarioMIP. The process of models' data.

cal_vpd: calculate VPD from specific humidity (huss), temperature (tas), and pressure (ps) from LESFMIP/ScenarioMIP/LUMIP/LS3MIP.

process: merge all data to one NC file from 1960-2014 or 2015-2099 (SSP126/245/370/585).

mask.rar: the mask file of Indo-Gangetic, South Asia Countries in 0.25/1.25 degree resolution.

irrigation.csv: the irrigation data (canal, wells, blanks, etc.) from DES, India.


namelist_irrOFF.input/namelist_irrON.input: WRF namelists for irrigation on and off in India.
